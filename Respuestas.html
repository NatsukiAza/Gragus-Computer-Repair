Las ventajas que nos proporciona GitHub son el poder manejar archivos en un amplio grupo de personas sin generar tanto conflicto y dandole una perspectiva a cada uno por separado que despues pueda agregar a un main

Definiciones

Un repositorio es un servidor de almacenamiento en el que se guardan varias ramas de un proyecto y puede ser modificado por varias personas.

Un git es un software creado con el proposito de crontolar varias versiones de un archivo o proyecto

Una rama es una division que sucede desde una parte anterior del repositorio siendo la principal main

Un conflicto es un problema que sucede cuando se quiere combinar 2 ramas pero el codigo de un archivo es muy similar al de otro por lo tanto se deja decidir manualmente que hacer con este conflicto
